# Datasets

This repository **does not redistribute** third‑party datasets. You must obtain datasets from their **official sources** and comply with their licenses/terms.

## Supported datasets (edit this list)

### 1) CO3D (restricted / terms‑governed)
- Official source: **Meta Research CO3D** (follow their access/terms and download instructions)
- This repo provides:
  - `scripts/download_co3d.sh` (prints official steps + validates local path)
  - `scripts/preprocess_co3d.py` (creates the processed layout expected by training)
  - `configs/data/co3d.yaml` (paths + preprocessing options)

**Expected layout**

```text
data/
  co3d/
    raw/         # your downloaded CO3D files (not committed)
    processed/   # generated by preprocess_co3d.py (not committed)
```

## Quick start (template)

1) Download the dataset from the official source (manual acceptance may be required):
```bash
bash scripts/download_co3d.sh --help
```

2) Set `DATA_ROOT` and preprocess:
```bash
python scripts/preprocess_co3d.py   --raw_root data/co3d/raw   --out_root data/co3d/processed   --splits configs/data/co3d.yaml
```

3) Train:
```bash
python -m bayesfusionsdf.train   --config configs/train.yaml   --data_root data/co3d/processed   --output_dir outputs/exp01
```

## Notes
- Do **not** commit `data/**` (already in `.gitignore`).
- If you add more datasets, follow the same pattern:
  - `scripts/download_<dataset>.sh|.py`
  - `scripts/preprocess_<dataset>.py`
  - `configs/data/<dataset>.yaml`
